# Android Clone Coding Project #1 - BMI 계산기
공식 = 몸무게(kg) / (키 (m) * 키 (m))
</br>
18.5 이하면 저체중
</br>
18.5 ~ 23 은 정상체중
</br>
23 ~ 25는 과체중
</br>
25 ~ 30 은 경도 비만
</br>
30 ~ 35 는 중정도비만
</br>
35 이상은 고도 비만
</br>
